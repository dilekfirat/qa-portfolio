# Test Cases Overview

Detailed test cases for this project are maintained in Excel format.

## Available Test Case Files
- Login test cases – `test-cases/Login_Test_Cases.xlsx`
- Product test cases – `test-cases/Product_Test_Cases.xlsx`
- Cart test cases – `test-cases/Cart_Test_Cases.xlsx`
- Checkout test cases – `test-cases/Checkout_Test_Cases.xlsx`

## Execution
- Test execution results are recorded directly in the test case files using
  **Actual Result** and **Status** columns.
- Failed test cases may result in defect reports.

## Defect Management
- Defects identified during test execution are documented separately in
  `manual-testing/bug-reports.md`.
