# Test Plan - SauceDemo WebApp

## 1. Introduction
This test plan contains the testing strategy, scope, resources and schedule for testing for SauceDemo WebApp (https://www.saucedemo.com/)
This document serves as a reference for manual testing activities before automation is introduced.

## 2. Scope

### 2.1 In Scope
- User Login
- Product Listing and Sorting
- Shopping Cart Functionality
- Checkout Process

### 2.2 Out of Scope
- Performance Testing
- Security Testing
- Accessibility Testing

## 3. Test Types
- Functional Testing
- Smoke Testing
- Regression Testing
- Exploratory Testing

## 4. Test Environment
- Application URL: https://www.saucedemo.com/
- Browsers: Google Chrome Version 143.0.7499.110, Safari Version 17.6
- Operating System: macOs Monterey Version 12.7.6
- Test Data: Accepted SauceDemo Users
